# HR Analytics Dashboard 📊

## 📌 Overview
This project is a Tableau dashboard analyzing employee attrition and workforce trends.

---

## 📊 Key Metrics
- Total Employees: 1470
- Attrition Count: 237
- Attrition Rate: 16.12%
- Active Employees: 1233

---

## 📈 Features
- Department-wise attrition analysis
- Age group distribution
- Job satisfaction insights
- Gender-based attrition analysis
- Education field impact

---

## 🛠 Tools Used
- Tableau
- Excel

---

## 🔍 Insights
- Sales department has highest attrition
- Age group 25–34 shows highest turnover
- Male attrition is slightly higher
